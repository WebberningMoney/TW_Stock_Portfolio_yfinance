# 台股庫存、損益與配息管理 v2.2

v2.2 延續 v2.1 的 yfinance＋Yahoo 台灣股利政策混合資料架構，重點改善股利資料重建、參數語意、下載效能、Yahoo 欄位映射，以及「庫存與損益」頁面的可用空間。

## v2.2 更新

### 股利資料依範圍清除後重建

按下「更新持股股利資料」後，程式會：

1. 依目前選擇的來源清除既有資料。
2. 依設定的 `1y／2y／3y／5y／10y／max` 範圍重新抓取。
3. yfinance 與 Yahoo 台灣爬蟲共用同一時間範圍。
4. 完成後整合兩來源的重複事件。

> 下載失敗時，該來源的舊資料已被清除，因此程式會先顯示確認視窗。

### 參數依資料流程分組

「抓取參數／單筆測試」現在分成：

- A. 商品清冊建立
- B. 商品行情下載
- C. 持股股利／股票分割同步
- D. 通用重試與容錯

每個參數都會標示影響的功能，不再使用語意不清楚的「歷史公司行動期間」。

### 效能改善

- 行情預設每批 100 檔，批次間隔降低。
- 行情失敗項目使用最多 4 個工作執行緒平行重試。
- yfinance 股利／股票分割改用批次 `download(actions=True)`。
- Yahoo 台灣股利政策爬蟲使用 2～4 個低度並行工作。
- API 與爬蟲仍保留逐項重試及退避機制。

### Yahoo 台灣欄位對齊

股利爬蟲依 Yahoo 頁面欄位順序解析：

- APP「除息日／分割事件日」← Yahoo「除息日」
- APP「現金股利發放日」← Yahoo「現金股利發放日」

已加入嘉里大榮 `2608.TW` 測試：除息日 `2025-06-13`、現金股利發放日 `2025-07-10`。

### 庫存與損益 UI

- 持股輸入區改為單列，釋放垂直空間。
- Yahoo Symbol 在庫存表隱藏，但仍作為內部識別值。
- 新增持股搜尋與顯示筆數。
- AI 研究工作區可收合／展開。
- 雙擊持股仍可快速進入修改流程。

## 建議參數

- 行情每批商品：100
- 行情執行緒：8
- 股利 API 每批持股：30
- 股利 API 執行緒：6
- 爬蟲同時抓取：3
- 爬蟲單檔間隔：0.35 秒
- 股利抓取範圍：5y
- 重試次數：3

## 升級舊資料

將舊版：

```text
data/portfolio_yfinance.db
```

複製到 v2.2 的 `data/` 即可。若要沿用抓取參數，也可複製 `data/app_settings.json`；舊版缺少的新參數會自動套用預設值。

## 資料限制

- yfinance 與 Yahoo 台灣頁面都不是台灣交易所正式結算資料。
- Yahoo 台灣爬蟲可能因網站改版需要調整。
- 股利金額與入帳狀態仍應以發行公司、投信及券商資料為準。
